# UNP_PROJECT_BI_WITH_POWER_BI_PIZZA_DATA_ANALYSIS-

I completed this Power BI project in collaboration with UNP Education as part of my analytics training. The dashboard analyzes pizza sales data to evaluate order trends, delivery performance, and delay patterns. It includes time-based insights such as peak hours and monthly demand fluctuations. Product analysis covers pizza size, type, and toppings to assess operational impact. The project demonstrates practical skills in data transformation, modeling, and interactive visualization.
